# Talent Fly International

Singapore-based talent and recruitment company website. Resume screening, shortlisting, and quality candidate placement for companies in India and abroad.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **react-markdown** (for policy and terms content)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000).

3. Build for production:
   ```bash
   npm run build
   npm start
   ```

## Project Structure

- `app/` – Routes and pages (marketing, policies, terms, documentation, immigration)
- `components/` – Header, Footer, LegalLayout
- `content/` – Markdown for policies (`content/policies/*.md`) and terms (`content/terms/*.md`)
- `lib/` – Content helpers (`content.ts`), news data (`news.ts`)

## Editing Content

- **Policies:** Edit or add `.md` files in `content/policies/`. Use slug as filename (e.g. `privacy.md`). Optional first line: `Last updated: DD Month YYYY`.
- **Terms:** Edit or add `.md` files in `content/terms/` (e.g. `general.md`).
- **Immigration news:** Edit `lib/news.ts` to add or update news items.
